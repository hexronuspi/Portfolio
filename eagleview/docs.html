    Overview

    eagleview is meant to provide easy visualisation of datasets, related to images.
    It is a package for easy installation, customization, and visualisation of images. 

    It is available as a pypi project hence it needs to be installed manually, either by cloning the repository on GitHub or using the `pip` command.

    The structure

    Python

    1. Display image number has maximum limit of matrix size. 
                        
                    !pip install eagleview

                    import eagleview

                    from eagleview.figshow import ImageMatrix
                    
                    image_display = ImageMatrix('/content/path_to_folder_containing_images')
                    image_display.display_image((2, 3), False)
    2. Display image number has no maximum limit and prints all images adding extra rows(number of column does not change). 
    
                        !pip install eagleview
    
                        import eagleview
                        
                        from eagleview.figshow import ImageMatrix
                        
                        image_display = ImageMatrix('/content/path_to_folder_containing_images')
                        image_display.display_image((2, 3), True)
                      # image_display.display_image((2, 3))

      2. 
                        !git clone https://github.com/hexronuspi/eagleview

    2. Use the PyPi project to directly import the files.
      3. 
                        !pip install eagleview
                        import eagleview
    After the package is successfully installed and imported it in your project, just run: 
    
                    from eagleview.figshow import ImageMatrix

                    image_display = ImageMatrix('/content/path_to_folder_containing_images')
                    image_display.display_image((2, 3), False)

    Options

    Display

    There are two display effects that you can use, restricted: which will not display more images than the size of matrix, and un-restricted: which will display all the images in the folder by adding more rows and preserving the columns.

    1. Display image number has maximum limit of matrix size. 
                            
                       
                        image_display.display_image((2, 3), False)
    2. Display image number has no maximum limit and prints all images adding extra rows(number of column does not change). 
    
                           
                            image_display.display_image((2, 3), True)
                        
    
                            image_display.display_image((2, 3))

    Additional Notes:

    Folder containing images

    If you have a compressed folder containing images, extract them and then run the code.

    Use GitHub cloning method

    The package version will be one version behind Github version for stable and smooth operating, for using the most recent beta test version, clone the GitHub repository and execute the program.
